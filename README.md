# util-array-diff

## Install
```bash
npm i --save @kldit/util-array-diff
```

## Usage
Load at the begining of your application.
```javascript
require("@kldit/util-array-diff");
```
Run at any where of your code.
```javascript
["banana", "apple", "guava"].diff(["banana"]) // ["apple", "guava"]
```
